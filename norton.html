Norton's theorm

Norton's Theorem states that any linear electrical network containing independent and dependent voltage and current sources can be replaced by an equivalent circuit consisting of a current source I N ​ in parallel with a resistor R N ​ . The Norton current ( I N ​ ) is the short-circuit current flowing through the output terminals, and the Norton resistance ( R N ​ ) is the equivalent resistance looking back into the circuit when all the independent sources are turned off. The Norton equivalent circuit is useful for simplifying circuit analysis, especially when dealing with the connection of multiple linear circuit elements.

To find the Norton equivalent circuit, you typically follow these steps:

Turn off all independent sources. This involves replacing voltage sources with short circuits and current sources with open circuits.
Find the short-circuit current ( I N ​ ). This is the current flowing between the output terminals of the circuit with the independent sources turned off.
Find the equivalent resistance ( R N ​ ). This is the resistance seen looking back into the circuit when all independent sources are turned off.
Once you have I N ​ and R N ​ , you can represent the original circuit by a Norton equivalent circuit with a current source I N ​ in parallel with a resistor R N ​ .
